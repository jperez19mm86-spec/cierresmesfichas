código · nombre · usuario · sistema · user_id · divisas · grupo_id · montos_rápidos.
Agrupa por código y suma las cajas. Re-importar actualiza (no duplica). Probá con Previsualizar antes.

📣 Avisos por Telegram

Al aceptar un pedido (Cargar), se avisa automáticamente al grupo de Telegram de ese cliente con el monto cargado. Configurá el bot una sola vez acá; el grupo (chatId) se pone en cada cliente (abrí un cliente → "📣 Grupo Telegram").

Creá el bot con @BotFather, agregalo a cada grupo de cliente, y poné el chatId del grupo en el cliente (botón "🔔 Probar" para verificar).

📥 Pedidos pendientes

Al apretar Cargar, el sistema entra a la página de la caja (usuario/contraseña → sesión) y carga las fichas pedidas. Rechazar descarta el pedido. Ambos quedan en el historial.
